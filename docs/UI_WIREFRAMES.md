## Design Notes

- **Color scheme:** Zinc palette, dark theme by default. Primary accent for interactive elements.
- **Typography:** System font stack, monospace for code/costs.
- **Responsive:** Sidebar collapses on mobile. Chat panels stack vertically.
- **Components:** Built with shadcn/ui (Button, Card, Input, Label, DropdownMenu, Avatar, Badge, Table, Tabs, Tooltip, Sonner for toasts).
- **Real-time:** WebSocket indicators for agent status (green dot = working, gray = idle, red = error). Toast notifications for budget alerts and agent events.
